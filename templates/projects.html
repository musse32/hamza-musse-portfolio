  • Spread of COVID-19 Deaths Across the 48 Contiguous States

    Data Source: mongodb+srv://readonly:readonly@covid-19.hip2i.mongodb.net/covid19

    Summary: I connected to a public MongoDB Cloud Atlas database containing data on COVID-19 collected by Johns's Hopkins University. I created a data pipeline using python to query the database (pymongo), clean the data, and transform to generate the cumulative sum of deaths for each state over time. This map represents the growing total death counts due to COVID-19 for the 48 Contiguous US States. Slide the top bar to show which states suffered the most death over the time period. The time period starts with the first recorded death in the US and is automatically updated every Sunday.


    Skills and tools: Python, MongoDB, HTML, Data Engineering, Data Visualization

  • 2048 Game: An exercise in Object-Oriented Programming

    Frontend Interface provided by FunHTMLGames.

  • Machine Learning Competition: Predicting Faulty Wells

    Summary: I participated in a Data Science competition hosted by DrivenData that challenged participants to use data about wells to predict whether water pumps were functional. Using data from Taarifa and the Tanzanian Ministry of Water, we predict which pumps are functional, which need some repairs, and which don't work at all.

    I implemented a Decision Tree Classifier to predict the three classes based on a number of variables about what kind of pump is operating, when it was installed, and how it is managed. A more detailed analysis along with the code is hosted on my Github.


    Skills and tools: Python, Pandas, Sci-Kit Learn, Statistical Analysis
